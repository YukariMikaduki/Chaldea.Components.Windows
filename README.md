## 自述  

此模块是为了方便日常开发工作而整合的工具集，包括但不限于：
- 常用的导出函数
	- `Chaldea.Components.DllImport` 命名空间
- 各类简化代码的扩展方法
	- `Chaldea.Components.Extensions` 命名空间
	- `Chaldea.Components.Utils` 命名空间
